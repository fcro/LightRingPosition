Principe du Jeu

Light Ring Position est un jeu qui consiste à gagner le maximum de cases possible en faisant attention aux obstacles.
Votre mission si vous l'accepter est de gagner la partie. Pour démarrer une partie vous devez être à plusieurs.
Vous avez la pouvez inviter vos amis à partir de notre application Android en leur ennvoyant un message.
Ou encore sur le site, sur votre mobile ou de votre tablette consulter la liste de vos amis et en ajouter de nouveaux.
Vous avez aussi la possibilité de d'échanger avec eux: le tchat est ouvert 24h sur 24 pour vous permettre de le faire.
Pour finir, vous êtes informés de toutes les parties qu'elles soient finies ou en attente avec les résultats obtenus.
Biensûr vous êtes libres de rejoindre une partie en attente quand vous le souhaitez chers visiteurs: vous en avez le droit.
